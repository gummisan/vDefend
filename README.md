# dfw-rule-stats.py
Usage: python3 dfw-rule-stats.py --server 10.1.1.20 --username admin -o test.csv



Retrieve NSX-T DFW Rule Statistics (Grouped).

options:
  -h, --help            show this help message and exit
  -s SERVER, --server SERVER
                        NSX-T Manager IP/FQDN
  -u USERNAME, --username USERNAME
                        NSX-T Username
  -p PASSWORD, --password PASSWORD
                        NSX-T Password (optional)
  -o OUTPUT, --output OUTPUT
                        Output CSV filename (e.g. stats.csv)
