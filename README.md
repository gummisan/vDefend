# dfw-rule-stats.py
Usage: python3 dfw-rule-stats.py --server 10.1.1.20 --username admin -o test.csv


Retrieve NSX DFW Rule Statistics (Grouped).
options:
  -h, --help show this help message and exit
  -s SERVER, --server <NSX Manager IP/FQDN>
  -u USERNAME, --username <NSX Username>
  -p PASSWORD, --password <NSX Password> (optional)
  -o OUTPUT, --output OUTPUT Output CSV filename (e.g. stats.csv)
